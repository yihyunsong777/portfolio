# 전화 알림 설정 가이드 (완전 무료!)

연락하기 폼 제출 시 스마트폰으로 즉시 알림을 받을 수 있습니다. **모두 무료**입니다!

## 방법 1: 텔레그램 봇 (추천 ⭐)

가장 간단하고 무료이며, 스마트폰 푸시 알림도 받을 수 있습니다.

### 설정 방법

1. **텔레그램 봇 생성**
   - 텔레그램 앱에서 `@BotFather` 검색
   - `/newbot` 명령어 전송
   - 봇 이름과 사용자명 설정
   - **봇 토큰** 복사해두기 (예: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

2. **채팅 ID 확인**
   - 텔레그램 앱에서 `@userinfobot` 검색
   - 봇과 대화 시작
   - **Your Id** 복사해두기 (예: `123456789`)

3. **코드에 적용**
   - `script.js` 파일 열기
   - 다음 부분 찾기:
     ```javascript
     const TELEGRAM_BOT_TOKEN = 'YOUR_TELEGRAM_BOT_TOKEN';
     const TELEGRAM_CHAT_ID = 'YOUR_TELEGRAM_CHAT_ID';
     ```
   - 각각 봇 토큰과 채팅 ID로 교체

### 테스트

1. 웹사이트의 연락하기 폼에 테스트 메시지 입력
2. 전송 버튼 클릭
3. 텔레그램 앱에서 즉시 알림 확인! 📱

---

## 방법 2: Discord 웹훅 (무료)

Discord를 사용하는 경우 이 방법도 좋습니다.

### 설정 방법

1. **Discord 웹훅 생성**
   - Discord 서버에서 채널 선택
   - 채널 설정(톱니바퀴) → 연동 → 웹후크
   - "웹후크 만들기" 클릭
   - 웹후크 이름 설정 (예: "포트폴리오 알림")
   - **웹후크 URL** 복사해두기

2. **코드에 적용**
   - `script.js` 파일 열기
   - 다음 부분 찾기:
     ```javascript
     const DISCORD_WEBHOOK_URL = 'YOUR_DISCORD_WEBHOOK_URL';
     ```
   - 웹훅 URL로 교체

### 테스트

1. 웹사이트의 연락하기 폼에 테스트 메시지 입력
2. 전송 버튼 클릭
3. Discord 채널에서 메시지 확인!

## 비용 안내

- ✅ **텔레그램 봇**: 완전 무료, 제한 없음
- ✅ **Discord 웹훅**: 완전 무료, 제한 없음
- ❌ **실제 SMS**: Twilio 등 유료 서비스 필요 (통화료 발생)

## 추천

**텔레그램 봇을 추천합니다!**
- 설정이 가장 간단
- 스마트폰 푸시 알림 자동 수신
- 완전 무료
- 제한 없음

## 보안 주의사항

⚠️ **중요**: 봇 토큰과 웹훅 URL은 GitHub에 공개되면 안 됩니다!

만약 GitHub에 푸시할 예정이라면:
1. `.gitignore`에 `script.js` 추가 (권장하지 않음)
2. 또는 환경 변수 사용 (복잡함)
3. 또는 봇 토큰을 간단히 암호화 (기본적인 보호)

개인 포트폴리오라면 공개되어도 큰 문제는 없지만, 봇 토큰이 노출되면 다른 사람이 봇을 사용할 수 있습니다. 필요시 봇을 삭제하고 새로 만들면 됩니다.
